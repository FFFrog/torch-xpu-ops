torch-xpu-ops*
===========================

torch-xpu-ops is an `xpu` implementation of PyTorch ATen operators.

## Build
* Standalone - Require pre-installation of PyTorch
```bash
mkdir build
cd build && cmake -DBUILD_TEST=1 -DPYTORCH_INSTALL_DIR=YOUR_PYTORCH_INSTALLATION_DIR ..
make -j x
```
